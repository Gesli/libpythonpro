# libpythonpro
Módulo para exemplificar construção de projetos Python no curso PyTools

Suportada a versão 3 de Python

Para instalar:

```console
python3 -m venv .venv
source . venv/Script/activate
pip install -r riquerements-dev.txt
```

Para conferir qualidade de código:
```console
flake8

```
